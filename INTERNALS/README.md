## Catholica / INTERNALS
Meta-Structural View

Catholic doctrine can be structurally reduced to four great arcs:

1. Exitus (Procession from God: Creation)
2. Lapsus (Fall into sin)
3. Redemptio (Return through Christ)
4. Consummatio (Eschatological fulfillment)

This exitus–reditus structure, articulated classically by Thomas Aquinas, integrates metaphysics, salvation history, sacramentality, and mystical theology into a single teleological architecture.
